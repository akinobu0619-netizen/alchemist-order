## 0. 一文サマリ

**拾った素材アイテムから幻獣を錬成できるようにし、「全てのドロップが幻獣への切符」になる素材経済を作る。**
錬金術がテーマのゲームなのに「物質から生命を創る」遊びがまだ無い——これはテーマの完成であり、後付けではない。

## 1. なぜやるか

- arkhomuで実証された最強の周回動機=「ドロップのガチャ券化」。素材を拾う→組み合わせを考える→錬成する、で探索が終わらなくなる
- AOの現状の弱点: 戦闘以外の獲得物が薄い（金と傷薬程度）。ダンジョン(Package D)の宝箱の中身が貧しい
- 既存の錬成(幻獣×幻獣)と同じUI文法で拡張できる=学習コスト最小
- **ホムンクルス(人造生命)はarkhomuの、幻獣錬成はAOの世界観**。仕組みは借りるが体験は錬金術書の文脈で出す

## 2. 素材アイテム体系（v1=24種で開始・700種は追わない）

```
素材 = { タイプ属性(9種のうち8) × 品位3段 } = 24種
  例: 火 → 「熾火の欠片」(並) / 「劫火の結晶」(良) / 「原初の火種」(極)
品位: 並(common) / 良(fine) / 極(prime)
```
- **入手**: 野生戦勝利drop(タイプ対応・並中心)、ダンジョン宝箱(良)、ヌシ討伐(極確定)、派遣調査(§7)、デイリー報酬
- **全素材に行き先を保証**(arkhomuの本質): 錬成の主素材/副素材、余剰は店売り(金策)。v2でもちもの合成
- データ実装: `src/game/materials.ts` に定義。`GameState.mats` を `Record<string, number>` へ拡張(旧talentStone/slotCharmは互換移行)

## 3. 素材錬成（アイテム→幻獣）

### レシピ構造
```
主素材(タイプ決定) + 副素材(品位=才能上限とレア枠) + フラスコ1 + 金
  → 主素材タイプの幻獣が誕生
```
| 副素材品位 | talent範囲 | レア種(stage2)混入率 | 変異種率 |
|---|---|---|---|
| 並 | 0-3 | 3% | 1/100(通常) |
| 良 | 2-6 | 10% | 1/100 |
| 極 | 4-8 | 25% + 予兆表示 | **1/50** |

- **予兆システム**(arkhomuの「予測」の移植): 錬成実行前に候補3種をシルエット+名前で提示。極素材使用時は確定枠1つ表示
  → 「回してみないと分からない」ではなく「狙って作る」感を担保
- 種プール: 主素材タイプの野生種+生成種(g-species)。**伝説は対象外**(伝説の格を守る)
- レベル: プレイヤーの最高Lv-3〜+0で生成(即戦力すぎない)
- コスト: フラスコ消費で捕獲資源と競合させる(「捕まえるか、創るか」の選択=ゲーム的意思決定)

### 演出(重要・手触り)
魔法陣に素材が吸い込まれ→光→誕生。既存の捕獲成功演出の資産を流用し、誕生時は図鑑登録と同格の扱い。

## 4. 乱数規約

`SPEC_RNG_REPLAY.md` §3に従い、素材錬成はfieldストリーム(非シード・systemRng)。将来の週塔とは無関係のため決定論不要。

## 5. バランスゲート（S3の轍を踏まない）

- 錬成産だけでジム勝率が10pt以上動かないこと(sim_balance.mjsに「錬成産talent分布パーティ」ケースを追加して確認)
- 素材dropは1戦闘平均0.6個以下(インフレ防止)。極は週数個レベルの希少性
- **受け入れテスト**: 30分プレイで並素材6-10個・良1-2個が自然に貯まり、「1回錬成してみる」に届くこと

## 6. ログインボーナス「師匠からの小包」（軽量・ローンチ前実装可）

- 日次初回起動時にダイアログ1枚:「師匠から小包が届いている」→ 開けると当日分を受取
- **連続日数の縛りなし・累計制**(罰でなくご褒美。スナック型ポジションに合わせ、途切れても失うものが無い)
- 報酬表(累計日数): 1日目=傷薬2 / 2=フラスコ2 / 3=並素材2 / 4=金300 / 5=良素材1 / 6=傷薬・上2 / **7=極素材1** → 以後7日周期でループ
- 実装: `lastLoginBonus: string(日付)` + `loginTotal: number` をGameStateへ。演出は既存の実績ダイアログ流用。工数極小
- 計測: `login_bonus` イベント(O1)

## 7. 派遣調査（遠征の軽量移植・M1後）

- ボックス(編成外)の幻獣を1体選んで派遣: 「森へ薬草採集」「火山へ鉱石採掘」等、行き先=世界に対応
- 所要: 実時間2時間/8時間の2択(タイムスタンプ保存、次回起動時に経過判定=**オフライン完結でサーバー不要**)
- 帰還報酬: 行き先タイプの素材(並2-3、低確率で良)+派遣した幻獣に少量exp
- 枠: 1(記章3個で2、6個で3に拡張=既存の進行度と接続)
- 狙い: **ボックスで眠る幻獣に仕事を与える**(収集の全個体に意味)＋数時間後の再訪アポ＋素材経済の蛇口
- 派遣中は錬成素材・パーティ編成に使えない(機会費用=選択)

## 8. 図鑑→入手導線（軽量・随時）

- 図鑑の未入手種に入手ヒントを表示: 「翠風の森に出現」「火の素材錬成で誕生することがある」
- 既入手種には「この種を素材錬成で狙う」ボタン(錬成画面へ主素材プリセットでジャンプ) — arkhomuの図鑑ジャンプUIの移植
