# 幻獣スプライト画像の置き場

ChatGPTなどで生成したモンスター画像を **このフォルダに `<id>.png` で保存**すると、
ゲーム内（バトル・図鑑・手持ち）の絵文字が自動でその画像に差し替わります。
（画像が無いidは絵文字のままなので、1体ずつ追加していけます。）

## 形式
- ファイル名 = 幻獣のid（例: イグニフ → `ignif.png`、ヴォルカドン → `volcadon.png`）
- 透過PNG・正方形（推奨 512×512 または 1024×1024）
- 1体を中央に全身、背景透明

## idの一覧
`MONSTER_ART_PROMPTS.md`（プロジェクト直下）に、全102体の id・名前・生成プロンプトを記載。
