# 幻獣スプライト画像の置き場

ChatGPTなどで生成したモンスター画像を **このフォルダに「図鑑番号.png」で保存**すると、
ゲーム内（バトル・図鑑・手持ち）の絵文字が自動でその画像に差し替わります。
（画像が無い番号は絵文字のままなので、1体ずつ追加していけます。）

## 形式
- ファイル名 = **図鑑番号（3桁ゼロ詰め）.png**（例: イグニフ=No.1 → `001.png`、ヴォルカドン=No.3 → `003.png`）
- ボスは `101.png`=灰王マグヌス、`102.png`=アビス・キメラ
- 透過PNG・正方形（推奨 512〜1254px四方）、1体を中央に全身
- **白背景のままでもOK**：`scripts/remove_bg.py` で透過化できます
  （例: `python scripts/remove_bg.py public/sprites/001.png`）

## 番号⇔幻獣の対応
`MONSTER_ART_PROMPTS.md`（プロジェクト直下）に、全102体の番号・名前・生成プロンプトを記載。
